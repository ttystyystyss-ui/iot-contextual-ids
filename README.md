# IoT Contextual IDS Packages

This repository stores the packaged artifacts for the IoT contextual IDS project.

## Files

- `gnn_gru_ids_package_20260513_201740.zip`: current GNN-GRU IDS code, trained model, metrics, validation predictions, online log, and dataset.
- `collection_scripts_package_20260513_201903.zip`: data collection and device control scripts.
